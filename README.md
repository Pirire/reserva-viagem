# Reserva de Viagem

Projeto backend para gerenciamento de reservas de viagem, integração com Stripe para pagamento e envio de e-mails via Nodemailer.

## Variáveis de ambiente

- STRIPE_SECRET_KEY
- STRIPE_WEBHOOK_SECRET
- MONGODB_URI
- FRONTEND_URL
- SMTP_HOST
- SMTP_PORT
- SMTP_SECURE
- SMTP_USER
- SMTP_PASS
- ADMIN_EMAIL

## Como rodar

```bash
npm install
npm start
